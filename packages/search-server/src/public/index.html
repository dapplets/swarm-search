Swarm Search

Search by metadata and content of files uploaded to Swarm.

Swarm Search API implements OpenSearch 1.1 Draft 6 interface.

It is fully open sorce and available at GitHub: https://github.com/dapplets/swarm-search/tree/main/packages/search-server

OpenSearch description document

Returns XML document describing the web interface of a search engine.

GET /opensearch.xml

RSS XML search results

Returns XML document with search results by a search query.

GET /rss?q={searchTerms}&count={count}&offset={startIndex}

HTML search results

Returns HTML document with search results by a search query.

GET /search?q={searchTerms}

JSON suggestions

Returns JSON document with suggestions by a search query.

GET /suggestions?part={searchTerms}
